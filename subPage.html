1.表示の仕組み

みんながwebページを調べてクリックした時にみえるこのWebサーバでは様々なアクセスするのにたくさんの処理が生じる。

その処理は難しい日本語が多くて見てても楽しくないので今回はその処理は説明しない！！

今回はこれらの小難しい処理の話ではなく、Webを書く、Webを作る方法を知ることが目的となる。

2.Webページの構造

プログラミング言語には上の図のようにたくさんの種類がある。

今回扱うのはHTMLとCSSとJavaScriptだ。HTMLとCSSは正確にはプログラミング言語ではないが、イメージとしては「Webを扱うプログラミング言語」である。

Webページは基本的にHTMLで作られている。

この言語の特徴は、タグを使った言語という点である。

タグとは<>のことである。この<>を使ってプログラミングするのがHTMLである。

どのように書くかというと< p>~ここに書きたいことを書く~< /p>といった感じ。

簡単に今回使うかもしれない文を書く基本のタグを紹介する。

< p>タグ　いまこれを書いているタグでもある。改行するのが特徴。

< h1>~< h6>タグ、以下のように大きさが変わるタグ

< h1>

< h2>

< h3>

< h4>

< h5>
< h6>

この７個を主に使うと思う。

3.実際に編集しよう

どこでもいいので右クリックして、検証をクリック

すると画面右にコンソール画面がでることを確認する

Stylesとhtmlと書いてある間の線をクリックしたまま下にもっていく

左の画像の◯のついている場所を左クリックして変えたい場所を左クリック

すると、画面の右側の部分にクリックした場所を書いているhtml(Webを扱うプログラミング言語)がでてくる

書き換えたいところをダブルクリックして書き換えよう

書き変わっていることを確認できたら終わり
